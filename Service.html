  • CATV & INTERNET

    SEMCO offers Cable TV and Internet Services.

    Cable TV provides a stable and reliable signal that is less susceptible to weather-related disruption. It offers a variety of channels (60 channels), which the whole family enjoys.

    Cable Internet provides speed and reliability at an affordable price, making it an excellent option for families with multiple devices, streaming, and gaming.

    Installation:

    CATV only - Php 1,000.00 Internet only - Php 2,500.00 Both CATV and Internet - Php 3,500.00

    Monthly Subscription Fee:

    Internet and CATV - Php 1,600.00 CATV only - Php 400.00 Internet only - Php 1,200.00 Additional TV Set per extension (Limited to 2 Extensions only) - Php 100.00

    Other Charges:

    Transfer Fee-CATV (Service Charge-Brgy Semirara) - Php 200.00 Transfer Fee-CATV (Service Charge-Bunlao Housing) - Php 500.00 Transfer Fee-INTERNET (Service Charge-Brgy Semirara) - Php 100.00 Reconnection Fee-CATV (Main Disconnected) - Php 300.00 Reconnection Fee-CATV (TV Extension) - Php 50.00 Reconnection Fee-INTERNET - Php 300.00 CATV Cable in excess of 15ft coaxial (cost per meter) - Php 35.00 Internet Cable in excess of 40m fiber optic (cost per meter) - Php 35.00 Replacement of Damaged Modem - Php 2,500.00
